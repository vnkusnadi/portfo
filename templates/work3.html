003 : Password Checker

My motivation for creating a password checker project stems from a concern for online security. While there are trusted websites that offer password APIs, such as https://haveibeenpwned.com/Passwords, it is essential to acknowledge the risks of relying solely on external platforms for a sensitive task. Even though certain websites may appear reputable, entrusting them with our passwords could expose our sensitive information and potentially lead to a data breach. Therefore, I have created this project to prioritize our own safety. Below is a password checker constructed using Python. 


            # before you execute this code, please install requests and hashlib library module in your cmd
            # execute this code on your chosen IDE and prepare a cmd
            # run: python {filename}.py {passwords you want to check}

            import requests
            import hashlib
            import sys

            # link for hash generator : https://passwordsgenerator.net/sha1-hash-generator/



            def request_api_data(query_char):
                # query_char =  first 5 digits of hash sha1 code of your password
                url = 'https://api.pwnedpasswords.com/range/' + query_char
                res = requests.get(url)

                if res.status_code != 200:
                    raise RuntimeError(
                        f'Error fetching: {res.status_code}, check the API and try again :)'
                else:
                    return res
                    # return the responese(200) where inside it gives list of password listed in data breach

            # hashes =  the list of
            def get_password_leaks_count(hashes, hash_to_check):
                li1 = []
                for line in hashes.text.splitlines():
                    li1.append(line.split(':'))
                for h, c in li1:
                    if h == hash_to_check:
                        return c


            def pwned_api_check(password):
                # convert password into sha1 hash
                sha1password = (hashlib.sha1(password.encode()).hexdigest()).upper()
                first5_letter, tail = sha1password[0:5], sha1password[5:]

                # response = the response of the web which is 200 (inside gives list of password in data breach)
                response = request_api_data(first5_letter)

                return get_password_leaks_count(response, tail)


            def main(args):
                for password in args:
                    count = pwned_api_check(password)
                    if count:
                        print(f'{password} was found {count} times....  you should change it')
                    else:
                        print(f'{password} was NOT found. Carry on!')
                return 'done!'

            if __name__ == '__main__':
                print(main(sys.argv[1:]))
