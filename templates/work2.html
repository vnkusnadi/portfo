002 : PDF merger

A PDF merger, commonly referred to as a PDF combiner, is a software utility designed to merge multiple PDF (Portable Document Format) files into a single, cohesive PDF document. The inception of this project was driven by the limitations frequently encountered when using online PDF merging services. These limitations often include the requirement for payment to access full features or frustrating restrictions, such as the inability to merge files exceeding 500 KB in size. This project serves as a complimentary and efficient solution, offering a free alternative. Below, is the Python code for the PDF merger solutions. 


            # before you execute this code, please install PyPDF2 library module in your cmd
            # execute this code on your chosen IDE and prepare a cmd
            # run: python {filename}.py {merged PDF name} {all PDF files}

            import sys
            import PyPDF2


            name = sys.argv[1]
            inputs = sys.argv[2:]

            def pdf_combiner(pdf_list):
                merger = PyPDF2.PdfFileMerger()
                for file in pdf_list:
                    merger.append(file)
                merger.write(name)
                print('PDF merged')

            pdf_combiner(inputs)
