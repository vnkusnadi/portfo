001 : Fringilla sit amet

In today's digital age, images play a great role in various application, including web development, graphic design, scientific research, and data analysis. Among the popular image formats, JPEG and PNG stand out. The motivation behind this project stems from PNG's unique strengths. PNG excels with its lossless compression, preserving image quality and intricate details. This feature proves invaluable, especially for images requiring a transparent background, like icons, logos, or web development assets. Since not all images come in PNG format, I initiated this project to bridge that gap. Below is the code to convert JPEG files ton PNG files 


            # execute this code on your chosen IDE and prepare a cmd
